## **Project Overview**

This project demonstrates an **end-to-end Machine Learning deployment workflow**:

- Train a model to predict medical insurance costs.
- Build a **Streamlit app** for interactive predictions.
- Containerize the app using **Docker** for easy deployment.
- Automate testing, building, and deployment using **GitHub Actions (CI/CD pipeline).**

---

## **Features**

- **Interactive Streamlit App**  
  Users can input personal details like age, BMI, smoking habits, and more to get insurance cost predictions.

- **Dockerized Application**  
  - The app runs in a container for consistent environment across systems.  
  - Easily deployable on cloud platforms or local machines.

- **CI/CD Pipeline**  
  - GitHub Actions automatically builds the Docker image on every push.  
  - Ensures continuous integration and deployment.  
  - Makes the app production-ready and maintainable.
